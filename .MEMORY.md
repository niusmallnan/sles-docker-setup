# Docker Pilot - Project Memory

## Project Overview

Docker Pilot is a Docker installation and TUI management tool specifically designed for SLES 15+. It provides:

- `config` - Interactive configuration wizard (quick/advanced mode) for registry, proxy, and container network CIDR
- `tui` - Built-in lazydocker for Docker management
- `scan` - CVE vulnerability scanning using embedded Trivy
- `ai-inspect` - Rule-based container health inspection

## Key Technical Details

- **Static binary**: Single file, zero external dependencies
- **Embedded tools**: lazydocker and trivy are embedded and extracted at runtime
- **Git history cleanup**: Large binaries removed from history with git-filter-repo
- **Pre-commit hook**: Auto-runs go fmt before commit
- **Test setup**: Makefile test-container uses docker:cli image with docker.sock mounted, and Trivy cache directory mapped
